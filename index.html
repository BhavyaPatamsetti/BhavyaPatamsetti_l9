GAME : Mix-Or-Match.

This website consists of a game.It's a memory game though.Which has a time limit of 100 seconds. In which you need to match cards.You will have to complete this within 100 seconds time limit. If you complete it you will find a pop-up with "VICTORY". If you lose you will need to replay. This is a simple game though.

1. You need to tap on "CLICK TO START".


2. You will be redirected to a screen which shows title of "Mix-Or-Match" and timer of 100 seconds.


3. You will be able to see 16 cards.


4. You need to select a card and again another card. If they match they will be opened and displayed.and try it until you match all these cards.


5. You will be poped up a note of "VICTORY" and "Click to restart".


6. If you fail to match all the 16 cards in time you will be poped up with "GAME OVER" and "REPLAY".
